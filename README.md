# Investment-Performance-Risk-Analytics-Dashboard-R-Shiny-
Shiny + shinydashboard performance reporting web app to analyze and compare investment portfolios across traditional funds, hedge funds, and benchmarks. Visualizes returns/risk, supports custom blended portfolios, and runs drawdown, Sharpe, benchmark tracking, and retirement ruin probability simulations.
Includes an end-to-end pipeline: Excel ingestion/cleaning (global.R), interactive UI (ui.R), and analytics server logic with plots/tables (server.R).
